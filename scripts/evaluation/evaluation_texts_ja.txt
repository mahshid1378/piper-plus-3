# 日本語評価用テストテキスト

# 基本的な挨拶
こんにちは、今日はいい天気ですね。
おはようございます。
ありがとうございました。
よろしくお願いします。

# アクセント評価
橋と箸と端の違いを説明します。
雨が降っても、飴を食べます。
花が咲いている端から、箸を使って食事をしました。

# 疑問文（異なるタイプ）
今日は何曜日ですか？
どこに行きたいですか？
本当にそうなんですか？
明日は晴れるでしょうか？

# 長文（自然な文章）
日本の四季は春夏秋冬がはっきりしており、それぞれの季節に独特の美しさがあります。春には桜が咲き、夏には蝉の声が響き、秋には紅葉が山を彩り、冬には雪が降り積もります。

# 感情表現
わあ、すごい！本当に嬉しいです！
申し訳ございません。
ええっ、まさか！
なんて素晴らしいんでしょう。

# 数字・固有名詞
電話番号は、03-1234-5678です。
2025年7月17日木曜日。
東京都渋谷区に住んでいます。
価格は1,980円です。

# 敬語・丁寧語
恐れ入りますが、もう一度お聞かせいただけますでしょうか。
お忙しいところ申し訳ございません。
ご確認のほど、よろしくお願いいたします。

# カタカナ語・外来語
コンピューターのアップデートが必要です。
レストランでディナーを楽しみました。
プレゼンテーションの準備をしています。

# 擬音語・擬態語
雨がザーザー降っています。
猫がニャーニャー鳴いています。
心臓がドキドキしています。

# 複雑な文構造
もし明日雨が降ったら、予定していたピクニックは中止にして、代わりに美術館に行くことにしましょう。
彼女が言っていたことは、つまり、私たちが考えていたよりも、ずっと複雑な問題だということです。
